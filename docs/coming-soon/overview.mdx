<Update label="Winter 2025" description="v1.0.0 (Core Agent)">
  - **Stabilization**: Core agent components are stable, ensuring reliability and predictable performance in production
  - **Destination Sinks**: Core Agent supports sending uniformly structured observability data to various common destination sinks
  - **Visualization**: Core Agent supports visualizing uniformly structured observability data
  - **Pre-compiled binaries**: Core agent binaries to make installation simpler
  - **On-the-fly configuration**: The Core Agent will be configurable at runtime without need for restart on each configuration
  - **Code coverage**: Expanding unit and integration test coverage in the Core Agent
  - **Discord**: Our designated community space for discussion, support, and collaboration on the platform
  - **Documentation update**: Updated docs, covering this release clearly and in detail for users
</Update>
